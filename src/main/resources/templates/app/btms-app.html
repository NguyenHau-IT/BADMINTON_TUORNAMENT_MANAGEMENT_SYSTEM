Yêu cầu Hệ thống

BTMS được tối ưu để chạy mượt mà trên các máy tính phổ thông. Kiểm tra yêu cầu hệ thống dưới đây trước khi tải xuống.

Hệ điều hành

Windows 10/11 (64-bit)

Bộ xử lý

Intel Core i3 hoặc tương đương

RAM

4 GB (khuyến nghị 8 GB)

Dung lượng

500 MB

Màn hình

1280x720 trở lên

Lưu ý quan trọng
  • Java Runtime tự động cài đặt cùng ứng dụng
  • Không cần cài đặt SQL Server riêng (embedded H2)
  • Tương thích với Windows Defender
  • Hỗ trợ nhiều màn hình (multi-monitor)
  • Offline mode - không cần internet để chạy
