BTMS (Badminton Tournament Management System) - Hệ thống quản lý giải đấu cầu lông chuyên nghiệp với Desktop App điều khiển 5 sân và Web Platform cập nhật real-time.
100% Miễn phí! Không giới hạn giải đấu, VĐV, thời gian sử dụng. Tất cả tính năng đều miễn phí vĩnh viễn.
OS: Windows 10/11 (64-bit)
RAM: 4GB+
CPU: Intel Core i3+
DB: SQL Server 2016+
Java: JRE 21+ (tự động cài đặt)
  1. Bật Server trong App
  2. Lấy mã PIN 4 số
  3. Mở browser nhập địa chỉ IP
  4. Nhập PIN → Bắt đầu cập nhật điểm

Dùng QR Code để truy cập nhanh!

Hỗ trợ 5 sân đồng thời:
  • Mỗi sân có bảng điện tử riêng full-screen
  • Mã PIN độc lập cho từng sân
  • Hỗ trợ đơn & đôi
  • Cập nhật real-time
